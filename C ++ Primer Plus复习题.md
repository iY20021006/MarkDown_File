# C ++ Primer Plus复习题

## 第二章 开始学习C++ 复习题

1. C++程序的模块叫什么？

   答案：叫作函数。

2. 下面的预处理编译指令有什么作用？

   `#include<iostream>`

​		答案：这将导致在最终的编译之前，使用` iostream `文件的内容替换该编译指令。

3. 下面的语句是做什么用的

   `using namespace std;`

​		答案：它使得程序可以使用 std 名称空间中的定义

4. 定义函数时，在什么情况不必使用关键字`return`？

​		答案：当函数的返回类型为 void 时，不用在函数中使用 return。然而，如果不提供返回值，则可以使 用它`return`
